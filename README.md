
# 🎬 Netflix User data Analytics

## 📌 Project Overview

This project analyzes Netflix user behavior data to uncover patterns in content preferences, subscription plans, audience demographics, watch time, and revenue generation. The objective is to transform raw user data into meaningful business insights that support better decision-making and improve customer engagement.

---

## 🎯 Objectives

- Analyze user viewing preferences across genres
- Identify revenue contribution by country
- Understand subscription plan distribution
- Compare audience segments (Adults vs Kids)
- Discover low-performing content categories
- Generate business insights using data visualization

---

## 📂 Dataset Features

The dataset contains information related to:

- User ID
- Country
- Favorite Genre
- Subscription Type
- Profile Type
- Watch Time
- Revenue
- Ratings
- Content Preferences

---

## 💡 Key Insights

- Drama and Action are among the most preferred genres.
- Adult users contribute the majority of platform engagement.
- UK, Australia, and India generate strong revenue.
- Subscription plans are distributed relatively evenly among users.
- Certain content categories show consistently lower ratings, indicating improvement opportunities.

---

## 📊 Dashboard Features

- KPI Cards for Total Users, Total Revenue, and Average Watch Time
- Favorite Genre Analysis
- Country-wise Revenue Comparison
- Subscription Plan Distribution
- Audience Split Analysis
- Lowest Rated Content Analysis
- Interactive Subscription Type Filter

---

## 🛠️ Tools and Technologies

- **Excel** – Data Cleaning and Initial Analysis
- **SQL** – Data Extraction and Querying
- **Python (Pandas, Matplotlib)** – Data Analysis and Visualization
- **Power BI** – Interactive Dashboard Development

---

## 🚀 Business Recommendations

- Invest more in high-engagement genres to maximize watch time.
- Create region-specific content strategies for high-revenue countries.
- Optimize subscription offerings based on user preferences.
- Improve low-rated content categories to enhance customer satisfaction.
- Leverage audience segmentation for personalized recommendations.
